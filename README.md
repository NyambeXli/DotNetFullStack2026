## 🌟 What is BookFlow?

BookFlow is a library management system built to **make book borrowing and management simple, fast, and reliable**.  
It allows users to browse books, check availability, and borrow them, while giving administrators tools to manage books, track circulation, and maintain high concurrency without slowing the system.  

---

## ✨ Key Features

* Real-time book availability and dynamic return tracking  
* Secure admin dashboard for managing books (CRUD + hide/unhide)  
* Responsive browsing with filtering for easy book discovery  
* High concurrency engine to handle multiple users at the same time  
* Automated reporting and circulation monitoring  

---

## 🛠️ Technical Skills Used

This project was built using:  

- **Backend:** C#, .NET Core, ASP.NET Core MVC  
- **Database:** SQL Server with Entity Framework Core  
- **Security:** JWT, ASP.NET Identity  
- **Messaging / Architecture:** RabbitMQ, Microservices concepts  
- **DevOps / Deployment:** Docker, Azure App Service, GitHub Actions (CI/CD)
